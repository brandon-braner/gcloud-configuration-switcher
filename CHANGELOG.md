# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-18

### Added

- **List Gcloud Configurations**: View all gcloud configurations with active status indicator
- **Switch Configuration**: Quickly activate any configuration from the list
- **Create Configuration**: Create new gcloud configurations with project, account, and region settings
- **Duplicate Configuration**: Clone existing configurations with all their properties
- **Delete Configuration**: Remove configurations with confirmation dialog
- **Refresh**: Reload configuration list on demand
